## Overview

The `useToast` hook provides a simple interface for displaying toast notifications in the FrontendPT application. It manages toast state and provides functions to show and hide toast messages with different types (success, error, info, etc.).

## Function Signature

```javascript
function useToast(): UseToastReturn
```

## Return Value

The hook returns an object with the following properties:

<ResponseField name="toast" type="object | null">
  Current toast object containing the message and type. Returns `null` when no toast is displayed.
  
  **Structure:**
  - `message`: The toast message string
  - `type`: The toast type (e.g., "success", "error", "info", "warning")
</ResponseField>

<ResponseField name="showToast" type="function">
  Function to display a new toast notification.
  
  **Signature:** `(message: string, type?: string) => void`
  
  **Parameters:**
  - `message`: The message to display in the toast (required)
  - `type`: The type of toast notification (optional, defaults to "success")
</ResponseField>

<ResponseField name="hideToast" type="function">
  Function to hide the current toast notification.
  
  **Signature:** `() => void`
</ResponseField>

## Implementation Details

<Note>
  The `useToast` hook uses `useCallback` to memoize the `showToast` and `hideToast` functions, ensuring stable function references across renders. This is particularly useful when passing these functions as props to child components.
</Note>

## Usage Example

### Basic Toast Usage

```javascript
import { useToast } from "@/application/hooks/useToast";

function MyComponent() {
  const { toast, showToast, hideToast } = useToast();

  const handleSuccess = () => {
    showToast("Operation completed successfully!", "success");
  };

  const handleError = () => {
    showToast("An error occurred. Please try again.", "error");
  };

  const handleInfo = () => {
    showToast("Here's some useful information.", "info");
  };

  return (
    <div>
      <button onClick={handleSuccess}>Show Success Toast</button>
      <button onClick={handleError}>Show Error Toast</button>
      <button onClick={handleInfo}>Show Info Toast</button>

      {toast && (
        <div className={`toast toast-${toast.type}`}>
          <p>{toast.message}</p>
          <button onClick={hideToast}>×</button>
        </div>
      )}
    </div>
  );
}
```

### Toast Component with Auto-Hide

```javascript
import { useToast } from "@/application/hooks/useToast";
import { useEffect } from "react";

function ToastContainer() {
  const { toast, hideToast } = useToast();

  useEffect(() => {
    if (toast) {
      // Auto-hide toast after 3 seconds
      const timer = setTimeout(() => {
        hideToast();
      }, 3000);

      return () => clearTimeout(timer);
    }
  }, [toast, hideToast]);

  if (!toast) return null;

  const toastStyles = {
    success: {
      backgroundColor: "#4caf50",
      color: "white",
    },
    error: {
      backgroundColor: "#f44336",
      color: "white",
    },
    info: {
      backgroundColor: "#2196f3",
      color: "white",
    },
    warning: {
      backgroundColor: "#ff9800",
      color: "white",
    },
  };

  return (
    <div
      style={{
        position: "fixed",
        bottom: "20px",
        right: "20px",
        padding: "16px 24px",
        borderRadius: "4px",
        boxShadow: "0 2px 8px rgba(0,0,0,0.2)",
        zIndex: 1000,
        ...toastStyles[toast.type],
      }}
    >
      <div style={{ display: "flex", alignItems: "center", gap: "12px" }}>
        <span>{toast.message}</span>
        <button
          onClick={hideToast}
          style={{
            background: "none",
            border: "none",
            color: "inherit",
            fontSize: "20px",
            cursor: "pointer",
          }}
        >
          ×
        </button>
      </div>
    </div>
  );
}
```

### Using Toast with Form Submission

```javascript
import { useToast } from "@/application/hooks/useToast";
import { useState } from "react";

function ProductForm() {
  const { showToast } = useToast();
  const [formData, setFormData] = useState({
    name: "",
    price: "",
    category: "",
  });

  const handleSubmit = async (e) => {
    e.preventDefault();

    try {
      // Submit form data
      const response = await fetch("/api/products", {
        method: "POST",
        headers: { "Content-Type": "application/json" },
        body: JSON.stringify(formData),
      });

      if (response.ok) {
        showToast("Product created successfully!", "success");
        // Reset form
        setFormData({ name: "", price: "", category: "" });
      } else {
        showToast("Failed to create product. Please try again.", "error");
      }
    } catch (error) {
      showToast("An unexpected error occurred.", "error");
    }
  };

  return (
    <form onSubmit={handleSubmit}>
      <input
        type="text"
        value={formData.name}
        onChange={(e) => setFormData({ ...formData, name: e.target.value })}
        placeholder="Product name"
        required
      />
      <input
        type="number"
        value={formData.price}
        onChange={(e) => setFormData({ ...formData, price: e.target.value })}
        placeholder="Price"
        required
      />
      <button type="submit">Create Product</button>
    </form>
  );
}
```

### Global Toast Provider Pattern

```javascript
import { useToast } from "@/application/hooks/useToast";
import { createContext, useContext } from "react";

const ToastContext = createContext(null);

export function ToastProvider({ children }) {
  const toast = useToast();

  return (
    <ToastContext.Provider value={toast}>
      {children}
      {toast.toast && (
        <div className={`global-toast toast-${toast.toast.type}`}>
          <p>{toast.toast.message}</p>
          <button onClick={toast.hideToast}>Close</button>
        </div>
      )}
    </ToastContext.Provider>
  );
}

// Custom hook to use toast context
export function useToastContext() {
  const context = useContext(ToastContext);
  if (!context) {
    throw new Error("useToastContext must be used within ToastProvider");
  }
  return context;
}

// Usage in components
function SomeComponent() {
  const { showToast } = useToastContext();

  const handleAction = () => {
    showToast("Action completed!", "success");
  };

  return <button onClick={handleAction}>Do Something</button>;
}
```

## Toast Types

While the hook accepts any string for the `type` parameter, common toast types include:

- `"success"` - For successful operations (default)
- `"error"` - For errors and failures
- `"info"` - For informational messages
- `"warning"` - For warning messages

The styling and behavior of these types should be implemented in your toast rendering component.
