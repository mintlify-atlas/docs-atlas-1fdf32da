<div className="mt-16 mb-16 max-w-5xl mx-auto px-6">
  <h2 className="text-3xl font-bold text-gray-900 dark:text-white mb-4">Quick start</h2>
  <p className="text-base text-gray-600 dark:text-gray-400 mb-8">
    Get your development environment up and running in minutes
  </p>
  
  <Steps>
    <Step title="Clone and install dependencies">
      Clone the repository and install all required packages.
      
      ```bash
      git clone https://github.com/Sunshide12/FrontendPT.git
      cd FrontendPT
      npm install
      ```
    </Step>
    
    <Step title="Configure the API endpoint">
      Set the backend API URL in `src/infrastructure/api.js`:
      
      ```javascript
      const api = axios.create({
        baseURL: "http://localhost:8000/api",
        headers: {
          "Content-Type": "application/json",
          Accept: "application/json",
        },
      });
      ```
      
      <Note>Make sure your Laravel backend is running on the specified port</Note>
    </Step>
    
    <Step title="Start the development server">
      Launch the Vite development server:
      
      ```bash
      npm run dev
      ```
      
      Your application will be available at `http://localhost:5173`
    </Step>
    
    <Step title="Build for production">
      When you're ready to deploy:
      
      ```bash
      npm run build
      ```
      
      The optimized build will be in the `dist` directory.
    </Step>
  </Steps>
</div>
