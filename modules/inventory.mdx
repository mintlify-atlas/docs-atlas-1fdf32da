## Overview

The Inventory module provides a focused interface for managing product stock levels. Unlike the Products module which handles all product data, the Inventory module is specifically designed for quick stock adjustments and monitoring.

### Hook Implementation

<Tabs>
  <Tab title="Hook Source">
    ```javascript src/application/hooks/useInventory.js
    import { useState, useEffect, useCallback } from "react";
    import * as inventoryService from "../../infrastructure/inventoryService";

    export function useInventory() {
      const [products, setProducts] = useState([]);
      const [pagination, setPagination] = useState(null);
      const [loading, setLoading] = useState(true);
      const [search, setSearch] = useState("");
      const [page, setPage] = useState(1);

      const fetchInventory = useCallback(async () => {
        try {
          setLoading(true);
          const data = await inventoryService.getInventory(page, search);
          setProducts(data.products.data);
          setPagination(data.products);
        } catch (err) {
          console.error("Error al cargar inventario", err);
        } finally {
          setLoading(false);
        }
      }, [page, search]);

      useEffect(() => {
        fetchInventory();
      }, [fetchInventory]);

      const adjust = async (id, stock) => {
        await inventoryService.adjustStock(id, stock);
        fetchInventory();
      };

      return {
        products,
        pagination,
        loading,
        search,
        setSearch,
        page,
        setPage,
        adjust,
      };
    }
    ```
  </Tab>
  <Tab title="Service Layer">
    ```javascript src/infrastructure/inventoryService.js
    import api from "./api";

    export const getInventory = async (page = 1, search = "") => {
      const params = { page };
      if (search) params.search = search;
      const { data } = await api.get("/inventory", { params });
      return data;
    };

    export const adjustStock = async (id, stock) => {
      const { data } = await api.patch(`/inventory/${id}`, { stock });
      return data.product;
    };
    ```
  </Tab>
</Tabs>

### Stock Adjustment Modal

```javascript
const [adjustingProduct, setAdjustingProduct] = useState(null);
const [tempStock, setTempStock] = useState("");

const handleAdjust = async () => {
  if (tempStock === "" || isNaN(tempStock) || Number(tempStock) < 0) return;
  try {
    await adjust(adjustingProduct.id, Number(tempStock));
    setAdjustingProduct(null);
    setTempStock("");
    showToast("Stock actualizado correctamente");
  } catch (err) {
    const msg = err.response?.data?.message ?? "Error al ajustar stock";
    showToast(msg, "error");
  }
};
```

### Modal UI Structure

```jsx
{adjustingProduct && (
  <Modal title={`Ajustar stock — ${adjustingProduct.name}`} onClose={() => setAdjustingProduct(null)}>
    <div className="flex flex-col gap-3">
      <p className="text-sm text-gray-500">
        Stock actual: <span className="font-semibold text-primary">{adjustingProduct.stock}</span>
      </p>
      <input 
        type="number" 
        min="0" 
        className="input" 
        placeholder="Nuevo stock" 
        value={tempStock} 
        onChange={e => setTempStock(e.target.value)}
        onKeyDown={e => e.key === "Enter" && handleAdjust()}
        autoFocus
      />
      <button onClick={handleAdjust}>
        Guardar
      </button>
    </div>
  </Modal>
)}
```

## Data Structure

### Inventory Product Object

```typescript
interface InventoryProduct {
  id: number;
  name: string;
  stock: number;
  category?: {
    id: number;
    name: string;
  };
}
```

<Note>
  The inventory endpoint returns product objects with category relationships populated.
</Note>

### API Endpoints

| Method | Endpoint | Parameters | Description |
|--------|----------|------------|-------------|
| GET | `/inventory` | `page`, `search` | Paginated product list with stock |
| PATCH | `/inventory/:id` | `{ stock: number }` | Update product stock |

## Validation Rules

<Warning>
  Stock adjustments must meet these criteria:
  - Value cannot be empty
  - Must be a valid number
  - Must be greater than or equal to 0
</Warning>

```javascript
const handleAdjust = async () => {
  // Validation checks
  if (tempStock === "") return;           // Not empty
  if (isNaN(tempStock)) return;          // Valid number
  if (Number(tempStock) < 0) return;     // Non-negative
  
  // Proceed with adjustment
  await adjust(adjustingProduct.id, Number(tempStock));
};
```

## Integration with Orders

The Inventory module is automatically updated when orders are processed:

- **Order Created**: Stock is reduced by order quantities
- **Order Cancelled**: Stock is restored (if order was pending)
- **Order Deleted**: Stock may be restored depending on order status

<Tip>
  The backend handles stock updates automatically during order operations. Manual adjustments through this module should be used for corrections, restocking, or damage.
</Tip>

## Keyboard Shortcuts

- **Enter**: Submit stock adjustment when modal is open

## Performance Optimizations

- Server-side pagination for large inventories
- Debounced search through useCallback dependencies
- Minimal re-renders with focused state updates
- Loading states prevent duplicate requests
