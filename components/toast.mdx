The Toast component displays temporary notification messages at the bottom-right of the screen. It automatically dismisses after 3 seconds and supports three visual variants: success, warning, and error.

## Visual Appearance

The Toast renders as:
- Fixed position at bottom-right corner
- Colored background and border based on type
- Icon indicator (✓, ⚠, ✕)
- Message text
- Close button (×)
- Smooth transitions and shadow
- Auto-dismiss after 3 seconds

## Props

<ParamField path="message" type="string" required>
  The notification message text to display
</ParamField>

<ParamField path="type" type="'success' | 'warning' | 'error'" default="success">
  The toast type, which determines the color scheme and icon:
  - `success`: Green background, checkmark icon (✓)
  - `warning`: Yellow background, warning icon (⚠)
  - `error`: Red background, error icon (✕)
</ParamField>

<ParamField path="onClose" type="function" required>
  Callback function invoked when the toast is dismissed (either automatically after 3s or manually via close button)
</ParamField>

## Usage Examples

### With useToast Hook

The recommended pattern is to use the `useToast` hook for managing toast state:

```jsx
import Toast from "../../components/Toast";
import { useToast } from "../../../application/hooks/useToast";

function ProductsPage() {
  const { toast, showToast, hideToast } = useToast();

  const handleCreateProduct = async (data) => {
    try {
      await createProduct(data);
      showToast("Producto creado con éxito", "success");
    } catch (error) {
      showToast("Error al crear el producto", "error");
    }
  };

  return (
    <div>
      {/* Page content */}
      
      {toast && (
        <Toast 
          message={toast.message} 
          type={toast.type} 
          onClose={hideToast} 
        />
      )}
    </div>
  );
}
```

### Manual State Management

```jsx
import Toast from "../../components/Toast";
import { useState } from "react";

function OrdersPage() {
  const [toast, setToast] = useState(null);

  const showSuccess = (message) => {
    setToast({ message, type: "success" });
  };

  const showError = (message) => {
    setToast({ message, type: "error" });
  };

  const hideToast = () => {
    setToast(null);
  };

  return (
    <div>
      {/* Page content */}
      
      {toast && (
        <Toast 
          message={toast.message} 
          type={toast.type} 
          onClose={hideToast} 
        />
      )}
    </div>
  );
}
```

## Auto-Dismiss Behavior

The Toast automatically dismisses after 3 seconds using a `useEffect` hook:

```jsx
useEffect(() => {
  const timer = setTimeout(onClose, 3000);
  return () => clearTimeout(timer);
}, [onClose]);
```

### Important Notes
- Timer starts when component mounts
- Timer is cleared if component unmounts early
- User can manually close before auto-dismiss
- Each new toast resets the timer

## Positioning

The Toast uses fixed positioning:
- Position: `fixed bottom-6 right-6`
- Z-index: 50 (appears above most content)
- Stays in corner even when scrolling

### Stacking Multiple Toasts

The current implementation shows one toast at a time. To stack multiple toasts, you would need to:

```jsx
const [toasts, setToasts] = useState([]);

const addToast = (message, type) => {
  const id = Date.now();
  setToasts(prev => [...prev, { id, message, type }]);
};

const removeToast = (id) => {
  setToasts(prev => prev.filter(t => t.id !== id));
};

return (
  <div>
    {toasts.map((toast, index) => (
      <Toast
        key={toast.id}
        message={toast.message}
        type={toast.type}
        onClose={() => removeToast(toast.id)}
        style={{ bottom: `${6 + index * 5}rem` }}
      />
    ))}
  </div>
);
```
