title: PageHeader
description: A reusable header component for displaying page titles, subtitles, and action buttons
---

The PageHeader component provides consistent page headers across all pages in the application. It displays a title, optional subtitle, and an optional action button or element.

## Visual Appearance

The PageHeader renders with:
- Large bold title (text-2xl)
- Optional smaller gray subtitle below the title
- Flexbox layout with space-between for title and action
- Bottom margin for spacing from content

## Props

<ParamField path="title" type="string" required>
  The main page title displayed prominently at the top
</ParamField>

<ParamField path="subtitle" type="string">
  Optional descriptive subtitle shown below the title in smaller, lighter text
</ParamField>

<ParamField path="action" type="ReactNode">
  Optional action element (typically a button) displayed on the right side of the header
</ParamField>

## Usage Examples

### Simple Header (Title and Subtitle Only)

```jsx
import PageHeader from "../../components/PageHeader";

function InventoryPage() {
  return (
    <div>
      <PageHeader 
        title="Inventario" 
        subtitle="Control de stock por producto" 
      />
      {/* Page content */}
    </div>
  );
}
```

### Header with Action Button

```jsx

function ProductsPage() {
  const openCreate = () => {
    // Open create modal
  };

  return (
    <div>
      <PageHeader

        subtitle="Catálogo y gestión de precios"

          >
            + Nuevo producto
          </button>
        }
      />
      {/* Page content */}
    </div>
  );
}
```

## Common Use Cases

- **List Pages**: Headers for product lists, client lists, order lists
- **Management Pages**: Headers for inventory, categories, and other management interfaces
- **Action Buttons**: "Create New" buttons positioned in the header
- **Page Identification**: Clear indication of current page context
- **Consistent Layout**: Uniform spacing and styling across all pages
